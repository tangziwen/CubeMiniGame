# CubeEngine/EngineSrc/2D

## Role

Retained 2D UI, sprites, labels, world billboards, notifications, ImGui bridge, and older GUI widgets.

## Important Objects

- `Sprite`: `Drawable2D` plus optional texture/color mesh and `EventListener` hit handling.
- `WorldBillboardSprite`: `Sprite` anchored to world position, projected to GUI coordinates, with owner/projected visibility.
- `WorldGuiProjector`: world-to-GUI and screen-to-GUI coordinate conversion helper.
- `LabelNew`: font/glyph-atlas text mesh drawable.
- `RetainedUISystem`: singleton retained UI root/layer manager; lazily finds or creates `gui_root` under the active scene.
- `NotificationSystem`: toast queue and retained notification views.
- `IMGUISystem`: ImGui input bridge, font setup, frame lifecycle, registered `IMGUIObject` draw calls, and capture state.
- `GUIWindowMgr`, `GUIStyleMgr`, `GUIFrame`, `GUITitledFrame`, `GUIRadioFrame`, `GUIListView`, `TableView`, `Button`, `BoxContainer`: older retained widget set.
- `NodeEditor`, `InspectableUI`: ImGui/editor helper surfaces.
- `SpriteInstanceMgr`, `TileMap2DMgr`: batched 2D-like world rendering helpers.
