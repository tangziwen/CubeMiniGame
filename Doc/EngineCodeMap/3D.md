# CubeEngine/EngineSrc/3D

## Role

3D scene objects and domain systems: models, primitives, terrain runtime, sky/shadow, particles, vegetation, and thumbnails.

## Important Objects

- `Model` / `ModelLoader`: multi-mesh drawable loaded from model JSON/cache, with material mapping and extra pose mesh lists.
- `SkyBox` / `Sky`: cubemap/atmosphere drawables and shared sky mesh/material resources.
- `ShadowMap`: CSM camera/projection/z-split helper and shadow map FBO access.
- `LinePrimitive`, `CubePrimitive`, `SpherePrimitive`, `CylinderPrimitive`, `LaserPrimitive`, `LineGrid`, `volumeMesh`: reusable primitive drawables.
- `ParticleSystem` / `ParticleEmitter` / emitter modules: particle update and emitter composition.
- `FoliageSystem`, `VegetationInfo`, `VegetationBatch`, `Foliage`: instanced vegetation registration, batching, LOD, and shadow submission.
- `Effect` / `EffectMgr`: effect resource manager used by older material/model paths.
- `Thumbnail`: offscreen snapshot target consumed by `GraphicsRenderer`.

## Terrain

- `TerrainRuntime`: owner/facade for terrain octree, mesh cache, sampler, mesher, drawable set, edit system, debug flags, view distance, and cache eviction settings.
- `TerrainOctree`, `TerrainOctreeNode`, `TerrainRenderSet`, `TerrainLodSelector`: LOD tree, render-cut selection, dirty marking, balancing, and node lookup.
- `TerrainMeshCache`, `TerrainMeshRequest`, `TerrainNeighborResolver`: mesh request identity, seam relations, ready/failed state, invalidation, eviction, and render-set freshness.
- `TerrainSampler`, `TerrainMesher`, `SurfaceNets`, `MarchingCubes`: sample voxel pages and build meshes.
- `TerrainDrawableSet`, `TerrainDrawableNode`: scene-facing drawables for ready terrain meshes.
- `TerrainEditSystem`, `GameMap`: voxel density/material edits, raycast, dirty bounds, and backing voxel data.
